# UW/WLU Double Degree GPA Calculator

A calculator for University of Waterloo and Wilfrid Laurier University double degree students to convert grades and calculate GPAs across both institutions.

## Features
- Convert between UW (0-100) and WLU (12-point) grading systems
- Calculate weighted averages for both universities
- Support for 4.0 GPA scale
- Add/remove courses dynamically

## Live Demo
[View the calculator here](https://vyviin.github.io/dd-calculator/)

```bash
npm install
npm run dev
```
